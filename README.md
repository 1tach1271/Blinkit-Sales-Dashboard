## Project Workflow

### Data Cleaning

- Used **Find & Replace** to standardize values.
- Checked missing values and duplicates using **Filters**.
- Filled missing numerical values using **Mean / Median**.

Excel made cleaning and preprocessing simple and efficient.

### Data Analysis

Created multiple Pivot Tables to analyze sales performance.

- **KPI Requirements** - Total Sales, Avg Sales, Number of Items, Avg Rating
- **Sales by Fat Content** - Compared Low Fat vs Regular
- **Fat Content by Outlet** - Compared sales across Tier 1, Tier 2, Tier 3
- **Sales by Item Type** - Identified top-performing product categories
- **Sales by Outlet Establishment** - Analyzed outlet performance by year
- **Sales by Outlet Size** - Compared High, Medium, Small outlets
- **Sales by Outlet Location** - Compared sales across tiers
- **Metrics by Outlet Type** - Compared sales, avg sales, and item count

### Dashboard

Built an interactive dashboard using:

- KPI Cards
- Charts
- Slicers
- Dynamic Filters

Excel helped in transforming raw data into meaningful business insights through easy analysis and visualization.

## Key Insights

- Total sales reached **$1.20M** from **8523 items**, with average sales of **$141** per item.
- **Low Fat products** contributed the highest sales (**65%**) compared to Regular products (**35%**).
- **Tier 3 outlets** generated the highest sales (**$472.1K**) among all outlet locations.
- **Medium-sized outlets** contributed the maximum sales (**42% of total sales**).
- **Fruits & Vegetables** and **Snack Foods** were the top-performing item categories.
